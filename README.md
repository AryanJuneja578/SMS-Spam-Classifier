# SMS-Spam-Classifier
A machine learning pipeline to classify SMS messages as spam or legitimate (ham) using NLP preprocessing and classical ML models.

## Overview

This project builds an end-to-end text classification system on the UCI SMS Spam Collection dataset. It includes a production-grade preprocessing module, TF-IDF vectorization, and multiple classifier comparisons.

## Pipeline

1. **Text Preprocessing** — URL/email/phone removal, lemmatization, stopword removal, special character cleaning
2. **Feature Extraction** — TF-IDF vectorization
3. **Classification** — Naive Bayes and Logistic Regression
4. **Evaluation** — Accuracy, F1-score, confusion matrix
